# Modern Login UI Design

A modern and responsive Login & Signup UI built using HTML, CSS, and JavaScript.

## 🚀 Live Features

- Modern Glassmorphism Design
- Responsive Layout
- Login Form
- Signup Form
- Forgot Password Functionality
- Email Validation
- Password Validation
- Show/Hide Password
- Dynamic Animations
- Smooth UI Effects
- Interactive Buttons

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
